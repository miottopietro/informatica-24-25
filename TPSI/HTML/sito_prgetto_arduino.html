Descrizione

Questo progetto utilizza un sensore ultrasonico per rilevare oggetti e misurare la distanza in modo preciso.

Componenti

  • Scheda Arduino Uno
  • Sensore ultrasonico HC-SR04
  • Servo motore
  • Breadboard
  • Cavi di collegamento

Codice

        
          // Esempio di codice Arduino
          #include <Servo.h>
          Servo servo;

          const int trigPin = 9;
          const int echoPin = 10;

          void setup() {
            servo.attach(11);
            pinMode(trigPin, OUTPUT);
            pinMode(echoPin, INPUT);
          }

          void loop() {
            for (int angle = 0; angle <= 180; angle += 10) {
              servo.write(angle);
              delay(500);
              
              long duration, distance;
              digitalWrite(trigPin, LOW);
              delayMicroseconds(2);
              digitalWrite(trigPin, HIGH);
              delayMicroseconds(10);
              digitalWrite(trigPin, LOW);

              duration = pulseIn(echoPin, HIGH);
              distance = (duration / 2) / 29.1;
